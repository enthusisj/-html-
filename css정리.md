# HTML와 CSS


## CSS(Cascading Style Sheets)
CSS는 HTML 구조에 디자인을 부여하는 도구이다.   
웹 페이지 구성요소의 스타일을 정의한는 언어이다.

-------------------------------------------
## CSS정리   
*font-family*   
: 글꼴을 설정하는 속성   

*margin*   
: 테두리(border)와 이웃하는 요소 사이의 간격인 마진 영역의 크기를 설정   

*padding*   
: 내용(content)과 테두리(border) 사이의 간격인 패딩 영역의 크기를 설정   

![logo](https://www.bitdegree.org/learn/storage/media/images/margin-css.o.png)    

*bold*   
: 굵은 굵기   

*lighter*   
: 상속된 값보다 얇은 굵기   

*border*   
: 테두리를 만드는 속성  

*solid*   
: 테두리를 실선으로 설정

*min-width*   
: 요소의 최소 너비를 지정   

*min-height*   
: 요소의 최소 높이를 지정   

*max-width*   
: 요소의 최대 너비를 지정

*max-height*   
: 요소의 최대 높이를 지정

*float*   
: 단어는 원래 '뜨다'라는 의미   
원래 웹페이지에서 이미지를 어떻게 띄워서 텍스트와 함께 배치할 것인가에 대한 속성   

*overflow*   
: 요소의 박스에 내용이 더 길 때 어떻게 보일지 선택하는 속성   

>overflow: visible, hidden, scroll, auto 차이   
* visible: 기본값으로 내용이 더 길어도 그대로 보인다. (내용이 흘러 넘침)
* hidden : 내용이 넘치면 자른다. 자른 부분은 보이지 않는다.
* scroll : 내용이 넘치지 않아도 항상 스크롤바가 보인다.
* auto : 내용이 잘릴 때만 스크롤 바가 보인다.   

*text-align*   
: 문서를 수평으로 정렬할 때 쓰인다.
